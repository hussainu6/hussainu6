# GitHub Achievements — Quick Unlock Guide

## Fast Unlocks (do today)

### 1. Pull Shark + Quickdraw + YOLO
- Open an **issue** in any of your repos
- Create a **branch**, fix it, open **PR**, merge within **5 min** → Quickdraw
- Merge **without review** → YOLO  
- Do **2 merged PRs** total → Pull Shark

### 2. Galaxy Brain
- Repo → **Settings** → **Features** → enable **Discussions**
- Create **Q&A** discussion, answer it, mark as **Solution**
- Do **2 times** → Galaxy Brain

### 3. Pair Extraordinaire
- Make a commit with: `Co-authored-by: Friend Name <friend@email.com>`
- Or: `git commit -m "msg" --author="Friend <friend@email.com>"`

### 4. Starstruck
- Any repo with **16+ stars**

### 5. Open Sourcerer
- Open PRs in **10+ different** public repos (typos, docs, small fixes)
